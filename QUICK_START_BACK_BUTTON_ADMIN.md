# Quick Start Guide - Back Button & Admin Features

## ✨ New Features Overview

### 1. **Back Button** 🔙
- Added to every page for easy navigation
- Click the **Back** button to return to the previous page
- Works across all sections: Dashboard, Admin, Reports, etc.

### 2. **Admin Dashboard Improvements**

The Admin Dashboard now has these fully functional pages:

#### **Manage Users** 👥
- View all registered users
- Filter by: All Users, Staff, Students, Admins
- See user statistics
- Pagination for easy browsing

#### **Reports & Analytics** 📊
- View complaint statistics
- See status breakdown (Pending, In Progress, Resolved, Rejected)
- View category distribution
- **NEW: Export to Excel** - Click buttons to export:
  - "Export All Complaints (Excel)"
  - "Export Statistics Report (Excel)"

#### **Recent Activity** 📝
- View latest complaints (10 most recent)
- See recent comments (15 most recent)
- Monitor system activity in real-time

#### **Assign to Staff** 👤
- Assign complaints to staff members
- Quick complaint summary
- Select from available staff

## 🚀 How to Use Each Feature

### Back Button
**Steps:**
1. Browse any page in the system
2. Click the **Back** button (with arrow icon)
3. You'll return to the previous page

**Available On:**
- Dashboard
- Complaint pages
- Admin pages
- Forms (Register, Login, Forgot Password, etc.)
- All other pages

---

### Accessing Admin Features

**To Access Admin Dashboard:**
1. Login as Staff or Admin user
2. Click **Admin** link in navigation bar
3. You'll see the Admin Dashboard

**From Admin Dashboard:**
- Click **Manage Users** - View all users and statistics
- Click **Reports** - View analytics and export reports
- Click **Activity** - See recent system activity

---

### Exporting Reports to Excel

**Step-by-Step:**

1. **Go to Reports Page**
   - From Admin Dashboard, click **Reports**

2. **Find Export Section**
   - Scroll down to "Export Reports" section
   - You'll see two export buttons

3. **Choose Export Type**
   - **Export All Complaints (Excel)** - Exports all complaint data
   - **Export Statistics Report (Excel)** - Exports summary + breakdown

4. **Download File**
   - Click desired button
   - File automatically downloads to your computer
   - Filename includes timestamp (e.g., `complaints_20260421_143022.xlsx`)

5. **Open in Excel**
   - Open downloaded file in Microsoft Excel or compatible software
   - Data is formatted with:
     - Professional headers
     - Proper column widths
     - Cell borders
     - Easy-to-read layout

## 🐛 Troubleshooting

### Back Button Not Working?
- Make sure JavaScript is enabled in your browser
- Try using browser's back button instead
- Clear browser cache if issues persist

### Can't Access Admin Pages?
- Make sure you're logged in as Staff or Admin
- Check that your account has admin/staff privileges
- Try logging out and logging back in

### Excel Export Not Working?
- Make sure you have Excel or compatible software
- Check browser's download folder
- Try a different browser if issues persist

### 404 Error on Admin Pages?
- This should no longer occur - all pages are properly configured
- If you see 404, please try:
  - Refreshing the page (Ctrl+F5)
  - Clearing browser cache
  - Logging out and back in
